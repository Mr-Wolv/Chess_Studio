# Third-Party Notices

## Stockfish 18

Chess Studio is designed to use Stockfish 18 as its UCI chess engine. Release builds may download and bundle the official Windows x86-64 Stockfish 18 binary from:

```text
https://github.com/official-stockfish/Stockfish/releases/download/sf_18/stockfish-windows-x86-64.zip
```

Stockfish is licensed under the GNU General Public License. When distributing Chess Studio with Stockfish bundled, include Stockfish's license and attribution requirements with the release package.

Official project:

```text
https://github.com/official-stockfish/Stockfish
```

## Python Dependencies

Chess Studio also depends on:

- `pygame`
- `python-chess` distributed as the `chess` package

Review dependency licenses before redistribution.
